# Prompt examples

Optional custom prompts can be stored on SD:

- `/app/StackChan/prompts/system.txt`
- `/app/StackChan/prompts/persona.txt`

Keep prompts short and local to your device. Avoid committing private names, home details, addresses, tokens, or internal server URLs.

Example persona:

```text
You are StackChan, a small embodied desktop robot. Be concise, friendly, and practical. Use the language the user speaks unless asked otherwise.
```

Example operational rule:

```text
During tool use, do not narrate intermediate steps unless the user asks. Speak only for clarification, permission, or final results.
```
